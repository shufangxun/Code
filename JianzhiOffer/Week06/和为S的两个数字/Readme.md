**题目**

输入一个数组和一个数字s，在数组中查找两个数，使得它们的和正好是s。

如果有多对数字的和等于s，输出任意一对即可。

可以认为每组输入中都至少含有一组满足条件的输出。

**样例**  
```
输入：[1,2,3,4] , sum=7

输出：[3,4]
```

**算法**

- 双指针法  
  排序之后, 两个左右指针, 逐渐内移  
  时间复杂度  
  排序 : O(nlogn), 双指针: O(n)

- hashmap法
