**题目**  

实现函数double Power(double base, int exponent)，求base的 exponent次方。

不得使用库函数，同时不需要考虑大数问题。

**注意：**

不会出现底数和指数同为0的情况

**样例1**
```
输入：10 ，2

输出：100
```
**样例2**  
```
输入：10 ，-2  

输出：0.01
```

**算法**
1. 分类讨论
- 底数为 0 
    - 指数为负 异常
    - 指数为 0, 输出 1
- 底数不为 0
    - 指数为正
    - 指数为负

2. 递归法  
考虑指数的奇偶性
