**题目**  
 
输入两个链表，找出它们的第一个公共结点。

当不存在公共节点时，返回空节点。

**样例**
给出两个链表如下所示：
```
A：        a1 → a2
                   ↘
                     c1 → c2 → c3
                   ↗            
B:     b1 → b2 → b3
```

输出第一个公共节点c1

**解法** 

1. 求差法  
分别从链表的头结点开始遍历，当两条链表有长度差时，先让长链表走他们的差值，当走到还剩下和短链表一样长时，两个链表同时遍历，这样就能找到第一个公共结点了

2. 栈  
构造两个栈，将链表压入栈，再出栈，当遇到第一个不相同的节点时，下一个就是第一个共同节点

3. 拼成一样长
不同部分为a和b，公共部分为c；a + c + b = b + c + a；让两个一起走，a走到头就转向b， b走到头转向a，则在公共部分相遇。
