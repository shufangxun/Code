**题目**
请实现两个函数，分别用来序列化和反序列化二叉树。

确保二叉树可以序列化为字符串，并且可以将此字符串反序列化为原始树结构。

**样例**  
序列化如下的二叉树
```python
    8
   / \
  12  2
     / \
    6   4

为："[8, 12, 2, null, null, 6, 4, null, null, null, null]"
```

**算法**  
可以使用前序遍历的方法来得到二叉树的序列，然后再每个节点之间得使用一个" ! "来隔开，这样可以避免节点值之间的歧义；对于空节点也需要存储下来，所以使用" # "来存储。

反序列化就解析序列化字符串即可。
