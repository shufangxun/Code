**Pytorch**

- torch.where 

  格式
  
  ```python
  # 如果是符合condition,输出a,否则输出b
  # 注意 a, b 都得为tensor
  torch.wherr(condition,a,b)
  ```
  
  用法
  
  ```python
  # 满足条件输出1,不满足输出0
  torch.where(a>2, torch.full_like(a,1), torch.full_like(a,0))
  # 第二种
  ones = torch.ones(a.shape)
  zeros = torch.zeros(a.shape)
  torch.where(a>2,ones,zeros)
  ```
