# 知识蒸馏

## 综述

关键问题

- 如何定义知识
- 用什么损失函数度量Teacher网络和Student网络的相似度

知识种类

- Output Transfer：将网络的输出作为知识
- Feature Transfer：将网络学习的特征作为知识，也就是中间层
- Relation Transfer：将网络或者样本的关系作为知识

损失函数种类

- CE
- L2 loss
