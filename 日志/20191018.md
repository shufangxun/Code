### **1. 刷题**   

**字符串中第一个出现频率一次的字符**  


常规做法 hash映射
```python
class Solution:
    def firstNotRepeatingChar(self, s):
        """
        :type s: str
        :rtype: str
        """
        d = dict()
        for i in range(len(s)):
            if s[i] not in d.keys():
                d[s[i]] = 1
            else:
                d[s[i]] += 1

        for i in range(len(s)):
            if d[s[i]] == 1:
                return s[i]

        return '#'
```

```python
class Solution:
    def firstNotRepeatingChar(self, s):
        """
        :type s: str
        :rtype: str
        """

        for i in range(0,len(s)):
            if s[i] not in s[:i]+s[i+1:]: # 精髓
                return s[i]
        return "#"
```
